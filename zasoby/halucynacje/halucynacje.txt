📚 OPIS DOŚWIADCZENIA

To praktyczne ćwiczenie pokazuje, jak różne parametry modelu AI wpływają na styl 
i charakter odpowiedzi. Używamy tego samego modelu bazowego (Bielik 4.5b), ale 
konfigurujemy go na dwa różne sposoby.


🔧 KONFIGURACJA MODELI

Model 1: BIELIK-URZEDNIK
├── Model bazowy: SpeakLeash/bielik-4.5b-v3.0-instruct:Q8_0
├── Temperatura: 0.1 (niska - deterministyczna, przewidywalna)
└── System prompt: "Jesteś precyzyjnym urzędnikiem. Odpowiadasz konkretnie i krótko."

Model 2: BIELIK-POETA
├── Model bazowy: SpeakLeash/bielik-4.5b-v3.0-instruct:Q8_0 (ten sam!)
├── Temperatura: 0.8 (wysoka - kreatywna, losowa)
└── System prompt: "Jesteś kreatywnym poetą. Uwielbiasz barwne opisy, rymy i metafory."


📝 PROMPT (identyczny dla obu modeli):
"Napisz o braku prądu w czwartek 10-12"

echo "FROM SpeakLeash/bielik-4.5b-v3.0-instruct:Q8_0" > TempModel && echo "PARAMETER temperature 0.1" >> TempModel && echo "SYSTEM Jesteś precyzyjnym urzędnikiem. Odpowiadasz konkretnie i krótko." >> TempModel && ollama create bielik-urzednik -f TempModel && rm TempModel

✓ Znacznie dłuższa i bardziej opisowa

🎯 TEMPERATURA - co to jest?

Temperatura to parametr kontrolujący "kreatywność" modelu:

• TEMPERATURA 0.0-0.3 (niska)
  → Przewidywalne, konserwatywne odpowiedzi
  → Wybiera najbardziej prawdopodobne słowa
  → Idealne dla: raportów, dokumentacji, odpowiedzi technicznych
  → Przykład: komunikaty urzędowe, instrukcje, FAQ

• TEMPERATURA 0.7-1.0 (wysoka)
  → Kreatywne, zróżnicowane odpowiedzi
  → Eksperymentuje ze słownictwem
  → Idealne dla: burzy mózgów, content marketingu, storytellingu
  → Przykład: posty na social media, opisy produktów, artykuły


💡 SYSTEM PROMPT - czemu jest ważny?

System prompt to "osobowość" AI - określa:
✓ Rolę (ekspert, asystent, nauczyciel)
✓ Ton wypowiedzi (formalny, przyjazny, techniczny)
✓ Styl komunikacji (zwięzły, opisowy, edukacyjny)
✓ Zachowanie (pomocny, krytyczny, kreatywny)


⚠️ "HALUCYNACJE" W AI - czym są?

Uwaga! Oba modele mogą "halucynować" - czyli generować informacje, które 
brzmią przekonująco, ale nie są prawdziwe.

W naszym przykładzie:
• Urzędnik napisał "w całym mieście" - informacja nieprawdziwa (nie było to w promptcie)
• Poeta dodał dramatyczne szczegóły o "pustych ulicach" - zmyślone

🔴 ZAWSZE WERYFIKUJ FAKTY generowane przez AI, szczególnie:
   - Daty, liczby, statystyki
   - Nazwy własne, miejsca, wydarzenia
   - Cytaty i referencje
   - Informacje techniczne lub medyczne
